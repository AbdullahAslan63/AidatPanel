# Resources

AidatPanel projesinin tüm dokümantasyon dosyaları bu klasörde toplanmıştır.

## Klasör Yapısı

- `raporlar_arshiv/` - Hata analiz raporları ve Faz 2 öncesi mükemmelleştirme raporu
- `analiz_raporlari/` - 6 farklı prompt analiz raporu (master, agents, görev dağılımı, aidatpanel, optimizasyon, güvenlik)
- `planning/` - Proje planlama ve yol haritası dosyaları
- `docs/` - Teknik dokümantasyon (backend endpoint analizi, mobile backend entegrasyonu)
- `checkpoints/` - Proje dönüm noktaları ve checkpoint kayıtları
- `AGENTS_COMPLIANCE_AUDIT.md` - AI agent uyumluluk denetimi
- `CHANGELOG.md` - Versiyon değişiklikleri ve changelog
- `YOL_HARITASI.md` - Proje yol haritası ve aşama planları
